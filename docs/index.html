Overview

Why ARIA?

Existing global PM₂.₅ products are either too coarse or lack forecasting capability. ARIA bridges this gap.

🌐

Truly Global

Trained on worldwide data — works equally over Europe, Asia, Africa, and the Americas.

🔬

1km Resolution

Far finer than operational CAMS forecasts. Captures urban gradients and topographic effects.

🏭

Emission-Aware

Integrates road networks, nighttime lights, and population density as structural emission priors.

⚡

Operational

Requires only ERA5 + CAMS at inference — both freely available from Copernicus.

📅

Multi-Horizon

Forecasts at J+1 to J+4 daily horizons. Supports near-real-time air quality monitoring.

🔓

Open Source

Full codebase and pretrained checkpoints on GitHub and HuggingFace.

Architecture

Cross-Resolution Attention

ARIA fuses global atmospheric context with fine-scale emission structure through a dual-branch Vision Transformer.

ERA5 (0.25°, 72ch) → Global Branch (ViT) ─────────────┐ 168×280 crop depth=8, dim=768 │ patch_size=8 → 735 tokens │ ├──→ Cross-Attention (2 layers) CAMS (12ch) + Elev (1ch) │ local queries ↔ global k/v Roads (1ch) + Lights (1ch) → Local Branch (ViT) ──────────┘ Population (1ch) + LatLon depth=6, dim=512 ↓ ── 18 channels total ── patch_size=16 → 1024 tokens CNN Decoder 32→64→128→256→512 ↓ PM₂.₅ @ 1km (delta prediction) Supervision: GHAP 0.01° · bf16 mixed precision · 128 GPUs · DDP Inference: ERA5 + CAMS + static layers only
