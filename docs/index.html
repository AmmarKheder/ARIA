Overview

Why ARIA?

Existing global PM₂.₅ products are either too coarse (25km) or lack forecasting capability. ARIA bridges this gap.

🌐

Truly Global

Covers the entire Earth — trained on worldwide data, not just regional subsets. Works equally well over Europe, Asia, Africa, and Americas.

🔬

1km Resolution

25× finer than CAMS operational forecasts (0.75°). Captures urban street-level pollution gradients and topographic effects.

🏭

Emission-Aware

Integrates road networks, nighttime lights, and population density as structural emission priors — first model to do so at global scale.

⚡

Fully Operational

Requires only ERA5 + CAMS at inference — both publicly available from Copernicus. No satellite imagery or ground stations needed.

📅

Multi-Horizon

Forecasts PM₂.₅ at J+1, J+2, J+3, J+4 daily horizons. Supports near-real-time air quality alerts worldwide.

🔓

Open Source

Full codebase, pretrained checkpoints, and inference API available on GitHub and HuggingFace. Reproducible and extensible.

Architecture

Cross-Resolution Attention

ARIA fuses global atmospheric context with fine-scale emission structure through a dual-branch Vision Transformer with cross-attention.

┌─────────────────────────────────────────────────────────────────┐ │ ARIA Architecture │ └─────────────────────────────────────────────────────────────────┘ ERA5 (0.25°, 72ch) → Global Branch (ViT-L) ──────────────┐ 168×280 crop depth=8, dim=768 │ patch_size=8 │ → 735 tokens │ ├──→ Cross-Attention CAMS (0.75°, 12ch) → ─────────────────────────────┐ │ (2 layers) Elevation (1ch) │ │ local → global Road density (1ch) → Local Branch (ViT-B) ──┘ │ Nighttime lights (1ch) depth=6, dim=512 │ Population (1ch) patch_size=16, 512×512 │ Lat/Lon (2ch) → 1024 tokens + skip │ ── 18 channels total ── │ ↓ CNN Decoder bilinear upsample 32→64→128→256→512 + skip connection ↓ PM₂.₅ @ 1km 512×512 patch delta prediction Training: GHAP supervision (0.01°) · bf16-mixed · 128 GPUs Inference: ERA5 + CAMS + static layers only (no GHAP needed)
