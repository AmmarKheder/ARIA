Overview

Why ARIA-Flow?

Existing global PM₂.₅ products are either too coarse or lack uncertainty quantification. ARIA-Flow bridges this gap with generative modeling.

🌐

Truly Global

Trained on worldwide data — works equally over Europe, Asia, Africa, and the Americas.

🔬

1km Resolution

Far finer than operational CAMS forecasts. Captures urban gradients and topographic effects.

🏭

Emission-Aware

Integrates OSM road networks, nighttime lights, and population density as structural emission priors.

🎲

Generative

Flow Matching produces probabilistic outputs. Run N trajectories for calibrated uncertainty estimates.

🔧

Counterfactual

Modify road density, then re-run the ODE. Test What-If emission scenarios for urban planning.

🔓

Open Source

Full codebase and pretrained checkpoints on GitHub. Inference requires only ERA5 + CAMS (free from Copernicus).

Architecture

Diffusion Transformer with AdaLN-Zero

ARIA-Flow fuses global atmospheric context with fine-scale emission structure through conditional flow matching and adaptive layer normalization.

ERA5 (0.25°, 72ch) → Global Branch (ViT) ─────────────┐ 168×280 crop depth=8, dim=768 │ patch_size=8 → 735 tokens │ ├──→ Cross-Attention (2 layers) CAMS (12ch) + Elev (1ch) │ local queries ↔ global k/v OSM Roads (1ch) + Lights → Local Branch (AdaLN-Zero) ────┘ Population (1ch) + LatLon depth=6, dim=512 ↓ + noisy x_t (1ch) patch_size=16 → 1024 tokens CNN Decoder ── 19 channels total ── time t conditions via AdaLN 32→64→128→256→512 ↓ v_θ velocity field (1, 512, 512) Flow Matching: x_t = (1-t)x_0 + t·x_1 → predict v_θ ≈ x_1 - x_0 Inference: 20 Euler ODE steps from N(0,I) → PM₂.₅ @ 1km Training: GHAP 0.01° · bf16 mixed · 128 MI250X GPUs · DDP
