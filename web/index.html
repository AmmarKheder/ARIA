Architecture

A dual-branch Vision Transformer with Conditional Flow Matching — the model learns straight-line ODE paths from noise to PM2.5 fields.

🌍

Global Branch

ERA5 meteorology (30ch) + CAMS chemistry (6ch) at 0.25° resolution. Patch size 8, 735 tokens.

Conditional Flow Matching

🎲

Linear ODE Path

xt = (1-t)x0 + t·x1 — straight paths from Gaussian noise to PM2.5 target. No complex diffusion schedules.

🧠

Velocity Prediction

The model predicts vθ ≈ x1 - x0, the velocity field that transports noise to clean PM2.5. AdaLN-Zero conditions on time t.

⚡

Fast Inference

Only 20 Euler ODE steps at inference — 5x faster than diffusion models. Ensemble sampling for uncertainty quantification.

Killer Feature

Counterfactual Urban Inference

Draw a new road. Erase a highway. Watch the atmosphere react in real-time.

ARIA-Flow learns the causal physics between high-resolution infrastructure and atmospheric chemistry. Urban planners can test What-If emission scenarios by modifying the road density tensor and observing the downstream PM2.5 response through the flow matching ODE.

1

Modify Road Density

Adjust highway traffic by -100% to +100%. The OSM-weighted road tensor updates in real-time.

2

Re-normalize

raw_new = clip(grid_total - grid_highway × α) → log1p(raw_new) / log1p(p99)

3

Run ODE

20 Euler steps through the learned velocity field → new PM2.5 map reflecting the counterfactual scenario.
